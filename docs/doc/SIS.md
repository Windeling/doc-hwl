# 学生信息查询系统使用文档

## 介绍
学生信息查询系统（SIS）是一个基于网页的工具，用于查询学生的基本信息、班级归属、考试成绩等。系统支持按姓名或代码查询，支持认证机制以确保数据安全。查询结果以卡片式布局呈现，支持交互式查看详情。

**重要提示**：
- 使用前请阅读[必读文档](https://doc.gxqzez.cn)。
- 系统数据基于学校内部信息，仅供授权用户使用。请尊重隐私，不得用于非法目的。
- 如果您遇到问题，请检查网络连接或刷新页面。

## 系统认证
1. **进入系统**：打开网页后，您将看到认证界面。
2. **输入认证代码**：系统提供8个输入框，每个框输入一个字符（字母或数字，大写）。
   - 输入时自动跳转到下一个框。
   - 回退键（Backspace）可返回上一个框。
   - 输入完成后，系统自动验证代码。
3. **验证结果**：
   - 成功：进入搜索界面。
   - 失败：输入框边框变红，请重试或联系管理员获取有效代码。

**提示**：认证代码由管理员提供，用于控制访问权限（如查看身份证号、电话等敏感信息）。

## 搜索功能
认证成功后，进入搜索界面。顶部有两个标签页：**按姓名查询**（默认）和**代码查询**。

### 1. 按姓名查询
- **输入框**：输入学生姓名（支持部分匹配，如输入“张”可找到所有姓张的学生）。
- **建议功能**：输入时，系统实时显示匹配姓名建议（最多5个）。点击建议可自动填充并搜索。
- **搜索**：点击“查询”按钮或按Enter键。
- **历史记录**：下方显示最近搜索历史（最多10条）。点击历史项可快速重复搜索。点击🗑️图标清除历史。

### 2. 代码查询
- **切换标签**：点击“代码查询”标签。
- **输入框**：输入特定代码，支持多种查询类型。系统会根据输入长度自动提示支持的代码格式。
- **支持代码类型**（提示文本会动态显示）：
  - **姓名拼音缩写**（3-4字母）：如“ZSF”查询姓名拼音为“Zhang San Feng”的学生。
  - **生日查询**（4/6/8位数字）：
    - 4位：年份，如“2008”查询2008年出生学生。
    - 6位：年月，如“0802”查询2008年2月出生学生。
    - 8位：年月日，如“20080215”查询2008年2月15日出生学生。
  - **地区代码**（1-6位数字）：
    - 1位：大区，如“3”查询华东地区学生。
    - 2位：省份，如“44”查询广东省学生。
    - 4位：城市，如“4403”查询深圳市学生。
    - 6位：区县，如“440303”查询深圳市福田区学生。
  - **班级代码**（5位数字）：如“24012”查询2024级高一2班学生。
  - **完整学号**（4位数字）：如“2401”查询学号为2401的学生。
  - **QQ号**（5-11位数字）：查询匹配QQ号的学生。
- **搜索**：点击“查询”按钮或按Enter键。

**通用提示**：
- 输入框支持自动焦点和清空（聚焦时清空上一次输入）。
- 查询时显示“正在查询，请稍候...”加载提示。
- 如果无结果，显示“未找到匹配的学生信息”。

## 查询结果
结果显示在下方玻璃卡片中，支持单一/多个结果。

### 1. 单一学生结果
- **基本信息**（分多个卡片显示，根据权限可见）：
  - 姓名、性别、级数。
  - 学号（部分隐藏，如****）、户籍地、生日。
  - 团员信息（职位、入团日期、团员编号、电话、民族）——仅限授权用户。
  - 社团信息（社团、职位）——仅限授权用户。
  - QQ信息（QQ号、昵称、个性签名、QID、等级、邮箱、注册时间、头像）——仅限授权用户。
  - 学生类型——仅限授权用户。
  - 照片：如果可用，显示在卡片右侧。
- **考试成绩**（仅限授权用户）：
  - 简要总结：分科/总平均分、平均校次/班次。
  - 点击“详情”展开：
    - 趋势图：校次趋势（多科目线图）和总分&校次趋势。
    - 考试卡片列表：点击卡片显示详细表格（分数、年级均分、相对比例、校次、班次）。
- **所在班级**（倒序显示CLASS5到CLASS1）：
  - 每个班级显示名称、类型（文/理）、评级（普通/重点/梦班/网班）。
  - 点击“详情”展开：班主任、各科老师、同学列表（可点击查看他人详情，按生日排序）、平均生日&差距、生日分布散点图（点击点显示当月学生）。
- **返回**：如果从列表进入，显示“返回列表”按钮。

### 2. 多个学生结果
- 显示“找到X个匹配的学生”标题。
- **排序**：下拉菜单选择“按姓名/学号/生日”排序。
- **学生列表**：每个项显示姓名、学号（部分隐藏）。点击“详情”查看单一学生信息。
- **返回**：从单一结果返回时，使用“返回列表”按钮。

### 3. 无结果或错误
- 无结果：显示“未找到匹配的学生信息”。
- 错误：显示红色错误消息，如“查询失败，请重试”。

## 其他功能
- **查询提示**：初始时显示“请输入姓名”和支持查询示例列表。
- **响应式设计**：支持手机/平板，布局自动调整（横向滚动图表）。
- **隐私保护**：敏感信息（如完整身份证、电话）根据认证权限隐藏或部分显示。
- **历史管理**：搜索历史本地存储，点击🗑️清除。

## 注意事项
- **权限限制**：不同认证代码有不同查看权限（如考试成绩、电话）。如果某些信息不可见，请联系管理员升级代码。
- **数据准确性**：信息基于最新CSV数据，可能有延迟。生日等数据可能有修正。
- **禁止查询**：某些学生信息受限，如果点击无响应，即为禁止。
- **性能**：大范围查询（如整个省份）可能稍慢，请耐心等待。
- **反馈**：如有bug或建议，请通过文档链接反馈。

如果您有疑问，欢迎咨询管理员。享受使用！
